1. **Internal form variables** (LeaguesManager, LeagueSetupView): `roundForm`, `showRoundModal`, `editingRound`, `addRound()`, `removeRound()` — kept for code stability, not visible to users.

3. **match.round property**: API returns match data with round field for match internals. Separate concern from phase terminology.
